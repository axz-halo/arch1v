# Arch1v - Music Curation Platform

A React Native Expo app for music discovery and social sharing, inspired by Dieter Rams design philosophy.

## 🎵 Features Included

### ✅ Working Components
- **Header** - Clean, minimal navigation
- **Bottom Tab Navigation** - Wave, Stations, Charts, Profile
- **Cover Flow** - Horizontal scrolling vinyl-style album covers
- **Wave Feed** - Social feed of what users are listening to
- **Now Playing Player** - Full-screen modal with vinyl record design
- **Vinyl Aesthetics** - Circular album art with center holes, inspired by LP records
- **Dieter Rams Design** - Minimalist UI with #ff5500 accent color

### 🎯 Interactive Features
- Tap album covers to open Now Playing player
- Tab navigation between screens
- Vinyl record UI with play/pause controls
- Social wave feed with reactions
- Clean, responsive design

## 🚀 How to Run

1. **Copy this App.tsx file to your local project**
2. **Navigate to your project directory**
   ```bash
   cd /path/to/your/arch1v
   ```
3. **Install dependencies (if not done)**
   ```bash
   npx expo install react-dom react-native-web @expo/metro-runtime
   ```
4. **Run the app**
   ```bash
   npm run web
   ```
